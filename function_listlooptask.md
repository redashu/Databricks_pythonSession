# Python Programming Task

## Question

Write a Python program that performs the following tasks:

1. Take multiple number inputs from the user and store them in a list.
2. The list should not contain more than 7 numbers.
3. Create one function to calculate the sum of all numbers in the list.
4. Create another function to sort the numbers in ascending order.
5. Display:
	- The original list
	- The total sum of numbers
	- The sorted list

## Requirements

1. Use a list to store the numbers.
2. Use two separate functions:
	- One function for adding all numbers
	- One function for sorting the list
3. Make sure only numeric values are entered by the user.
4. The program should not allow more than 7 numbers in the list.

## Example Input

```text
Enter how many numbers (max 7): 5

Enter number 1: 12
Enter number 2: 5
Enter number 3: 20
Enter number 4: 8
Enter number 5: 15
```

## Expected Output

```text
Original List: [12, 5, 20, 8, 15]

Sum of Numbers: 60

Sorted List: [5, 8, 12, 15, 20]
```

## Additional Challenge (Optional)

Modify the program to:

1. Sort the list in descending order.
2. Find the largest and smallest number.
3. Remove duplicate values from the list.
4. Display the average of all numbers.
