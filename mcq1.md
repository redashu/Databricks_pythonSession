# MCQ 1

1. What is Python primarily classified as?
	- A. Markup language
	- B. Interpreted programming language
	- C. Database language
	- D. Operating system

2. Which software executes Python code line by line?
	- A. Compiler
	- B. Assembler
	- C. Interpreter
	- D. Linker

3. Which of the following is an integer data type in Python?
	- A. 10.5
	- B. "10"
	- C. 10
	- D. True

4. Which of the following represents a float value in Python?
	- A. 25
	- B. "25.7"
	- C. 25.7
	- D. False

5. Which of the following is used to create a string in Python?
	- A. []
	- B. {}
	- C. Single or double quotes
	- D. ()

6. What is the output data type of "Hello" in Python?
	- A. int
	- B. float
	- C. string
	- D. boolean

7. Which function is used to identify the data type of a variable in Python?
	- A. datatype()
	- B. type()
	- C. check()
	- D. id()

8. Which of the following is NOT a basic Python data type?
	- A. int
	- B. float
	- C. string
	- D. character

9. What will be the type of 5.0 in Python?
	- A. int
	- B. string
	- C. float
	- D. list

10. Which of the following is a valid string in Python?
	- A. Hello
	- B. "Hello"
	- C. 123Hello
	- D. @Hello

## Answers

1. B
2. C
3. C
4. C
5. C
6. C
7. B
8. D
9. C
10. B
