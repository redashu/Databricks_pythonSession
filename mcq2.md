# MCQ 2

1. Which collection type in Python is ordered and changeable?
	- A. set
	- B. tuple
	- C. list
	- D. dictionary

2. Which Python collection does NOT allow duplicate values?
	- A. list
	- B. tuple
	- C. dictionary
	- D. set

3. Which keyword is used for decision-making in Python?
	- A. loop
	- B. if
	- C. switch
	- D. case

4. Which loop is used to iterate over a sequence in Python?
	- A. repeat loop
	- B. do-while loop
	- C. for loop
	- D. foreach loop

5. Which symbol is used to define a dictionary in Python?
	- A. []
	- B. ()
	- C. {}
	- D. <>

6. What is the correct keyword to create a function in Python?
	- A. function
	- B. define
	- C. def
	- D. fun

7. Which collection type is immutable in Python?
	- A. list
	- B. set
	- C. dictionary
	- D. tuple

8. Which statement is used to stop a loop immediately in Python?
	- A. stop
	- B. exit
	- C. break
	- D. continue

9. What is the purpose of a function in Python?
	- A. To store files
	- B. To repeat a block of reusable code
	- C. To create variables only
	- D. To install packages

10. Which keyword is used to return a value from a function?
	 - A. print
	 - B. output
	 - C. send
	 - D. return
